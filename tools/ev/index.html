Evanced Formatting

Need to tweak the way your Evanced program or report looks. Use these bookmarklets to do just that!

To reformat reports:

Report Formatter

To reformat program listings:

Program Formatter

Help

  1. Click and drag the button up to your browser's bookmark or favorites bar.
  2. A bookmark will appear there called Report Formatter or Program Formatter.
  3. Run an Evenaced report or go to edit a program listing.
  4. Click the bookmark and viola!

What is it doing?

The bookmarklet is doing a very simple thing. It is injected a new set of rules for how the page should look into the page. That's it!
