How Do I?

Need to solve a problem quickly? Here are some of the common things people using the library need to do.

  • Apply for jobs at the Library
  • Contact the library
  • Donate to the library
  • Download ebooks or music
  • Get a library card
  • Get directions/parking info
  • Have a test/exam proctored
  • Renew items online
  • Register for events
  • Reserve public meeting rooms
  • Sign up for e-mail hold notifications
  • Sign up for the newsletter
  • Sign up for event notifications
  • Use the computers/Wi-Fi
  • View My Account
  • Volunteer at the library
